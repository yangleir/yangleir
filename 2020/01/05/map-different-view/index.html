竖版地图的科学绘制

发表于 2020-01-05 更新于 2021-02-25 分类于 GMT 阅读次数： Valine：

竖地图

竖版地图其实也是一种地图投影，国内的竖版地图由长期从事大地测量学、地球物理学、地震学、地图学、地理学、马克思主义哲学研究的诗人郝晓光研究员编制，他的竖版地图采用了和传统地图上北下南不一样的视角，展现了一个奇异有趣的世界格局，特别是提高了人们对极地的认知，将南北极以较小的变形投影到一张图上，科学考察的路线一目了然。

“经向世界地图”采用“等差分纬线多圆锥投影方法”，而我们所做的只是将“经向世界地图”的纵坐标线（经线）与横坐标线（纬线），变换成“纬向世界地图”的横坐标（经线）与纵坐标线（纬线）。也就是说，“纬向世界地图”与“经向世界地图”在投影方法上的区别仅仅是“坐标换轴”。

GMT绘制的Robinson投影全球地图

1963年，阿瑟·h·罗宾逊(Arthur H. Robinson)响应兰德·麦克纳利公司(Rand McNally company)的呼吁，发明了罗宾逊投影。从那时起，该公司就在通用世界地图中使用了这种投影技术。罗宾逊在1974年公布了该计划的建造细节，1988年，国家地理学会(NGS)开始使用罗宾逊投影法绘制通用世界地图，取代了范德格林滕投影法，NGS在1998年放弃了罗宾逊投影，转而使用Winkel tripel投影，因为后者“减少了靠近极地的陆地的变形”。来源

图投影的基本原理是把球面展开，通过一定的数学关系投影到平面。按照变形基本上又分为等面积投影、等角度投影和任意投影。传统地图照顾到人类的居住大陆的投影变形，而忽略了南北极的变形，且遵从上北下南的常识，而竖地图则考照顾到南北极的变形，一定程度上增加人类居住区域的变形，例如南美洲。如下图展示，正常地图是东西“撕开”球面，竖版地图则是一种换了方向的球面展开，是南北“撕开”球面。

南北“撕开”地球后，在按照一定的数学法则投影到平面上，即得竖版地图。郝晓光研究员所研发的竖版地图以开创性的视角迅速占据了国内外市场，获得了大众的喜爱。下图为其竖版地图的样貌，可见和常规地图比较，这种地图给人的感觉极具冲击性和颠覆性。竖版地图不仅视角独特，可以用于广大中小学的地图教学和科普，也可以在国家重大项目中发挥作用。例如利用竖版地图，可以使决策者对极地区域有更直观的位置认知，中美之间的最近距离不经过太平洋，而是经过北冰洋，那么中美一旦爆发冲突，北极区域的防御就尤为重要，在北极海域增加我国的战略考量尤为重要。

竖版地图，图片来源：郝晓光主页
